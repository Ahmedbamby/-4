# شركة رواد المكافحة — صفحة هبوط

صفحة هبوط ثابتة (Static) بالكامل، جاهزة للنشر على GitHub Pages.

## المحتويات
- `index.html` — الصفحة الكاملة (HTML + CSS + JS مضمّن).

## خطوات النشر على GitHub Pages

1. أنشئ مستودعًا جديدًا على GitHub ثم ارفع محتوى مجلد `docs/` (أو المشروع كاملًا).
2. من صفحة المستودع اذهب إلى **Settings → Pages**.
3. تحت **Source** اختر:
   - **Branch:** `main`
   - **Folder:** `/docs` (إذا رفعت المجلد كما هو)، أو `/root` إذا نقلت `index.html` إلى جذر المستودع.
4. اضغط **Save**، ثم انتظر دقيقة، وسيظهر رابط الموقع في نفس الصفحة.

## بديل: نشر من جذر المستودع
انسخ محتوى مجلد `docs/` إلى جذر المستودع، ثم اختر `/root` كمجلد للنشر.

## ملاحظات
- جميع الروابط نسبية، تعمل مباشرة على GitHub Pages بدون أي إعدادات إضافية.
- TikTok Pixel مضمّن داخل `<head>` مع تتبع نقرات أزرار واتساب والاتصال.
- لا يوجد Backend ولا قواعد بيانات — الصفحة تعمل كملف ثابت.
